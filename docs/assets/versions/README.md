# OneFlow asset versions

This folder preserves branded asset snapshots so earlier visual directions can be reused later.

- `2026-04-26-oneflow-before-social-preview-alignment` — snapshot taken before aligning the live social preview with the adoption-focused copy.
- `2026-04-26-oneflow-adoption-aligned` — snapshot taken after aligning the live social preview to `Simple and easy to adopt.`

- `2026-04-26-oneflow-o-symbol-wordmark` — snapshot taken after introducing the integrated `O` symbol and the horizontal flow line through the wordmark.

- `2026-04-26-oneflow-final-wordmark-candidate` — snapshot taken after the final polish pass to make the wordmark feel more like one typographic `OneFlow` lockup.
