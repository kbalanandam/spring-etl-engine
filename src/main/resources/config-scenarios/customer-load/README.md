# customer-load

Business scenario for loading only customer data.

## Flow

- source: CSV `Customers`
- target: XML `Customers`
- processor: default field-to-field mapping

## Files

- `job-config.yaml`
- `source-config.yaml`
- `target-config.yaml`
- `processor-config.yaml`

## Run example

Set `etl.config.job` to this scenario's `job-config.yaml`.
